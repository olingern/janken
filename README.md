# yaba

Short for yabai, "whoa" in Japanese.

**Version**: 0.0.1 (_hardly usable_)

A universal database terminal client. 


## Installation

```bash
npm install yaba
```

## Setup

```
yaba init
```

## Configuration

Yaba's configuration file is stored in your home directory under `.yaba`. Example:

```
/home/user/.yaba
```
